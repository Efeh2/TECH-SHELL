# TECH-SHELL Store

A modern, responsive React + Vite storefront for the Tech Shell brand.

## Live Preview

- Website: https://tech-shell.vercel.app/

## Overview

TECH-SHELL is an ecommerce homepage built with React and Vite. The store features a hero section, product categories, shop navigation, and a cart page for a polished shopping experience.

## Key Features

- Responsive layout
- Smooth navigation
- Product listing and detail pages
- Cart functionality
- Modern UI styling

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Local Setup

1. Install dependencies

```bash
npm install
```

2. Start development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```
